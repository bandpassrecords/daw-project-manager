# DAW Project Manager Website

This is the static website for DAW Project Manager, hosted on GitHub Pages.

## Setup

1. The website is automatically deployed to GitHub Pages from the `gh-pages` branch
2. Make sure GitHub Pages is enabled in your repository settings
3. Point GitHub Pages to the `gh-pages` branch

## Files

- `index.html` - Main landing page
- `styles.css` - All styling
- `script.js` - JavaScript for smooth scrolling and animations

## Customization

### Update GitHub Links

Replace `bandpassrecords` in `index.html` with your actual GitHub username:
- Search for: `bandpassrecords/daw_project_manager`
- Replace with: `YOUR_USERNAME/daw_project_manager`

### Update Download Links

Update the download links in the download section to point to your actual release assets.

### Add Screenshots

You can add a screenshots section by:
1. Adding screenshots to a `screenshots/` folder
2. Adding a new section in `index.html` to display them

## Deployment

The website is automatically deployed when you push to the `gh-pages` branch. GitHub Pages will serve the files from the root of this branch.
